staticpackage

Validates the git parameter based on a regex expression to determine both validity, as well as if it is a URL or repo name.

Parameters
gitParamThe git parameter passed by the user.
Returns
Enumeration of the validation result.

If the parameter is valid and a URL, returns GitParamType.URL If the parameter is valid and a repo name, returns GitParamType.REPO_NAME If the parameter is in invalid form, returns GitParamType.INVALID
